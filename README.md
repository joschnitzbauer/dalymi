# dalymi: *data like you mean it*

*dalymi* is a lightweight, data-focused and non-opinionated pipeline manager written in and for Python.

## Installation
### Stable
```
pip install dalymi
```

### Latest development
```
git clone https://github.com/joschnitzbauer/dalymi.git
cd dalymi
pip install .
```
